Computer Club Syllabus

What you will learn:


You will learn how to program in HTML, XML, Lua, Javascript,-making, and other fun activities, website making and more!

Note: No homework will be given because this is a club.

This includes Codecademy, GitHub for website making, Roblox Studio for Lua, and learning how to type faster to improve your WPM

You will be learning more later in the school year, suggestions are always welcome
